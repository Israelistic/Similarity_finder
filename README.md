# Similarity_finder


## Usage

Create a csv that looks like this:

| image1 | image2 |
|:------:|:------:|
| img1.png|img2.png|

Browse the cvs and click submit CSV
And the output file will look something like:

| image1 | image2 | similar | elapsed |
|:------:|:------:|:-------:|:-------:|
|img1.png|img2.png|0.56|255|
